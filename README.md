### Description
SMC-001 보드를 Arduino Uno 및 DFR CAN Shield를 이용하여 제어하는 예제코드 모음.  
* Mode 0: 펄스기반 위치제어
* Mode 1: 지속 회전 속도제어
* Mode 2: 엔코더 기반 위치제어

### CAN Header File 준비
DFR CAN Shield 제공 라이브러리 [다운로드][1] or "mcp_can" 폴더 사용.  
"mcp_can" 폴더는 /Documents/Aruduino/libraries 폴더 내에 넣어놓기.
